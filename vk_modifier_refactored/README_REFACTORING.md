# VK Track Modifier — Рефакторинг проекта

## Структура проекта

```
vk_modifier/
├── __init__.py           # Инициализация пакета, версия
├── main.py               # Точка входа приложения
├── config.py             # Конфигурация приложения
├── constants.py          # Константы и настройки
├── vk_modifier_config.json  # Пользовательская конфигурация
│
├── core/                 # Ядро приложения (бизнес-логика)
│   ├── __init__.py
│   ├── ffmpeg.py         # Обёртка над ffmpeg: поиск, запуск, probe
│   ├── filters.py        # Аудио фильтры и эффекты
│   ├── branding.py       # Брендирование и метаданные
│   ├── track_info.py     # Информация о треке
│   ├── worker.py         # Воркер для обработки треков
│   └── presets.py        # Пресеты настроек
│
├── ui/                   # Пользовательский интерфейс
│   ├── __init__.py
│   ├── styles.py         # Стили UI компонентов
│   ├── widgets.py        # Кастомные виджеты
│   ├── daw_interface.py  # Главное окно DAW-интерфейса
│   │
│   └── tabs/             # Вкладки интерфейса
│       ├── __init__.py
│       ├── audio_tab.py      # Вкладка аудио эффектов
│       ├── structure_tab.py  # Вкладка структуры (обрезка, вырезка)
│       ├── metadata_tab.py   # Вкладка метаданных и брендирования
│       └── export_tab.py     # Вкладка экспорта и настроек
│
└── resources/            # Ресурсы (ffmpeg, обложки, иконки)
    └── (пустая папка для будущих ресурсов)
```

## Изменения в рефакторинге

### ✅ Удалённые лишние элементы:
- Файл `ui/styles.py` отсутствовал, но на него были ссылки — **создан заново** с необходимыми стилями
- Лог-файл `vk_modifier.log` не копируется (генерируется при запуске)

### ✅ Сохранённый полезный функционал:
- **Все вкладки интерфейса**: Audio, Structure, Metadata, Export
- **Кастомные виджеты**: GlassCard, SliderWithLabel, WaveformWidget, FileListWidget, CoverPreviewLabel
- **DAW-интерфейс**: Полный стиль из asdas.html с современными градиентами
- **Методы обработки**: Обрезка, вырезка, сращивание, фильтрация, брендирование
- **Антидетект функции**: Сдвиг фреймов, модификация длительности, случайный битрейт

### ✅ Улучшения структуры:
1. **Чёткое разделение**: core (логика) / ui (интерфейс) / resources (ресурсы)
2. **Централизованные стили**: `ui/styles.py` вместо дублирования в разных файлах
3. **Пакетная структура**: Правильные `__init__.py` во всех папках
4. **Чистые импорты**: Относительные импорты внутри пакета

## Запуск приложения

```bash
cd vk_modifier_refactored
python -m vk_modifier.main
```

## Требования

- Python 3.9+
- PyQt5
- ffmpeg (должен быть в PATH или в папке resources)

## Примечания

- Все исходные файлы сохранены в `/workspace/vk_modifier/`
- Рефакторированная версия находится в `/workspace/vk_modifier_refactored/`
- Логи больше не включаются в репозиторий (добавьте `*.log` в `.gitignore`)
